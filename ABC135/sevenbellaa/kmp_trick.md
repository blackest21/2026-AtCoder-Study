# KMP 이어붙이기 트릭

## 핵심 아이디어

> **패턴 `t`와 텍스트 `s`를 구분자 `#`으로 이어붙여서, failure function 한 번으로 전처리와 매칭을 동시에 처리한다.**

## 구조

```
text = t + '#' + s

       [  t (패턴)  ] [#] [   s (본문)   ]
       ← 전처리 구간 →     ← 매칭 구간 →
```

- **`t` 구간** (`i = 0 ~ |t|-1`): 패턴 자체의 failure 배열이 구축된다.
- **`#`** (`i = |t|`): failure를 0으로 리셋하는 분리벽. `s`와 `t`에 등장하지 않는 문자를 사용한다.
- **`s` 구간** (`i = |t|+1 ~`): `t`의 failure 정보를 활용하여 매칭이 진행된다. `F[i] == |t|`이면 해당 위치에서 `t`의 완전 매칭이 발생한 것이다.

## `#` (구분자)의 역할

`#`이 없으면 `t`의 접미사와 `s`의 접두사가 이어져서 **가짜 매칭**이 발생할 수 있다.

`#`은 어떤 문자와도 매칭되지 않으므로, `F[|t|] = 0`이 보장되며, `s` 구간에서의 `F[i]` 값이 `|t|`를 **절대 초과하지 못한다**.

## 구현

```cpp
// failure function (KMP 전처리)
auto failure(string_view sv) {
  vector<int> f(sv.size());
  for (int i = 1, j = 0; i < (int)sv.size(); i++) {
    while (j > 0 && sv[i] != sv[j]) j = f[j - 1];
    if (sv[i] == sv[j]) f[i] = ++j;
  }
  return f;
}

// s에서 t의 모든 출현 위치 찾기
string text = t + '#' + s;
auto F = failure(text);
int tn = t.size();

for (int i = tn + 1; i < (int)text.size(); i++) {
  if (F[i] == tn) {
    int pos = i - tn + 1 - (tn + 1); // s에서의 시작 위치
    // pos 위치에서 t가 매칭됨
  }
}
```

## 일반 KMP와 비교

|  | 일반 KMP (2단계) | 이어붙이기 트릭 (1단계) |
| --- | --- | --- |
| 전처리 | `failure(t)` | `failure(t + '#' + s)` |
| 매칭 | 별도의 매칭 루프 필요 | failure 결과에서 `F[i] == \|t\|` 만 확인 |
| 시간복잡도 | O(\|s\| + \|t\|) | O(\|s\| + \|t\|) |
| 코드 길이 | 길다 | **짧다** |

## 예시

```
t = "aba",  s = "abababac"

text = "aba#abababac"
index:  0123456789...

F =    [0,0,1,0,1,2,3,2,3,2,3,0]
                      ^     ^
                    F[6]=3  F[8]=3  (3 == |t|)

→ 매칭 위치:
  i=6: pos = 6 - 3 + 1 - 4 = 0   ← s[0..2] = "aba" ✓
  i=8: pos = 8 - 3 + 1 - 4 = 2   ← s[2..4] = "aba" ✓
```

## 응용

- **기본 패턴 매칭**: 위의 예시 그대로
- **순환 문자열에서 매칭**: `text = t + '#' + s + s` (s를 두 번 이어붙임)
- **무한 반복 문자열에서 매칭**: `text = t + '#' + s × k` (s를 충분히 반복)
